Summary

At TU Dortmund, I contributed to human-drone interaction and drone swarm research. The work combined data generation, control tuning, sensor fusion and robotics software migration.

Key contributions

  • Contributed to a 6DoF IMU dataset for training drone swarm gesture-control models.
  • Improved localization accuracy using PID tuning and EKF sensor fusion.
  • Integrated IR camera and IMU data for automated battery charging during swarming operations.
  • Contributed to migration of a ROS-based swarm-show platform toward ROS2 using Crazyswarm.
  • Fine-tuned Crazyflie drone controllers to improve stability for pallet detection with a camera deck.
  • Supported and troubleshot drone shows for indoor logistics applications.

Recruiter takeaway

This work shows robotics system thinking: sensing, estimation, control, software migration and practical demo readiness.
