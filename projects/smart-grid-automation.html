Summary

This project focused on smart grid automation and standard-compliant data workflows. The work connected Intelligent Electronic Device models, IEC 61850 configuration and CIM-based automation.

Key contributions

  • Refined flexibility modeling of IEDs according to IEC 61850 standards.
  • Upgraded CIMPY 2.4 to CIMPY 3.0 for IEC 61970-301 compatibility.
  • Developed an automated framework for generating IID files using ICD templates.
  • Built a Python framework for Digital Grid Connection Points and household auto-configuration .scd files.
